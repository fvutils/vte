Reference

Command Reference

usage: vte [-h] {generate,preprocess,list,quickstart} ...

Positional Arguments

subcmd

Possible choices: generate, preprocess, list, quickstart

Sub-commands:

generate

generate source files

vte generate [-h] [-force] template name
Positional Arguments
template

ID of template

name

Name to use in the template

Named Arguments
-force

force overwrite of existing files

Default: False

preprocess

Pre-process a file

vte preprocess [-h] [-o OUTPUT] file
Positional Arguments
file

Input file

Named Arguments
-o, --output

Specify output

list

list available templates

vte list [-h]

quickstart

Creates an initial .vte template-descriptor file

vte quickstart [-h] [-o OUTDIR]
Named Arguments
-o, --outdir

Specifies the output directory (default: cwd)
